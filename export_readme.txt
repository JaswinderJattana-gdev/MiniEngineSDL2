MechShooter Level 01

How to run:
1. Open this folder.
2. Double-click MechShooter_Level01.exe.

Do not move the exe away from the assets folder.
The game expects this structure:

MechShooter_Level01/
  MechShooter_Level01.exe
  SDL2.dll
  assets/
    levels/
      level01.txt

Controls:
WASD / Arrow Keys - Move
Mouse - Aim turret
Space - Fire
Shift - Dash
P - Pause
ESC - Menu
F1 - Debug overlay
